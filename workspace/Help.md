# Help

## Flow

- Add Node/Edit Node
    - Add Node with Label
    - Edit Label and Image

## Markdown

- Add Note
    - Add Markdown file
- Add Comment/Edit Comment
    - Comment on selected text

    - Table Format

## Shortcut

- Ctrl+o
    - Open Workspace Dialog
- Ctrl+q
    - Close the application
- Ctrl+n
    - Switch to Flow
- Ctrl+m
    - Switch to Markdown
- Ctrl+1
    - Switch to Preview Only
- Ctrl+2
    - Switch to Markdown Only
- Ctrl+3
    - Switch to Both
- Ctrl+w
    - Show/Hide Comments
- Ctrl+e
    - Show/Hide Edit Node
- F1
    - Help
